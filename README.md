## Installation  
1. Clone this repository:
   ```bash
   git clone https://github.com/Alirezasedd/AI-NPC-Dialogue.git
   ```
2. Navigate to the project folder:
   ```bash
   cd AI-NPC-Dialogue
   ```
3. Install dependencies:
   ```bash
   pip install openai textblob pyttsx3
   ```
4. Run the script:
   ```bash
   python npc_dialogue.py
   ```
